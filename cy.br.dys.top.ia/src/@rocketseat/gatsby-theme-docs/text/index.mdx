# cy.br/dys.top.ia RPG - Introduction

Welcome to the cy.br/dys.top.ia !

## WHAT IS THIS?

This is a tabletop RPG. It's meant to be played around a table with friends. Each should have a device capable of browsing this website (smartphone, tablet, notebook, etc). And dice (this game uses d%s).

### website sections

Your character lives in the topmost section: See the STATS and TRAITS pages.
